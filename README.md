# eloquent-app-practice

## 概要

COACHTECH 教材 Tutorial 9-4「Eloquent ORM ハンズオン演習」で作成した成果物です。
Eloquentモデルを使ってブログシステム投稿一覧、作成、編集ページをつくりました。

## 使用技術

- PHP 8.x
- Laravel 10.x
- Eloquent ORM
- MySQL
- GitHub

## 学んだこと

- モデルでの属性の設定の仕方を学べました。-　テストデータの作成でthinkerとSeederの違いを学べました。
- コントローラーでのCRUD操作について学べました。
